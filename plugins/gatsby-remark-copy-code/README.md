# gatsby-remark-copy-code

Add buttons to **markdown** code snippets. 

## Install

```bash
npm install gatsby-remark-copy-code --save-dev
```

## How to use

in your `gatsby-config.js`

```js
plugins: [
  {
    resolve: 'gatsby-transformer-remark',
    options: {
      plugins: ['gatsby-remark-copy-code']
    }
  }
]
```
